github 仓库的建立 ​

首先建立一个github的账号 网址是 https://github.com/


仓库的创建 ​

名称最好使用自己的用户名后面还要打上 .github.io


未来将自己在电脑上写的代码同步到github创建的仓库中要提前安装 git bash x64位的电脑安装Git/x64的安装包


密钥的获取 ​

密钥可以将自己的电脑和github 账号联系在一起后面就可以通过git bash 将 代码上传

密钥的获取方式 ​

首先在本地文件夹里面新创建一个文件夹命名为 key 打开 git bash 唤出控制面板



输入 ssh-keygen -t rsa -C "输入注册github时使用的邮箱"注意ssh-keygen 之间没有空格而且C要大写


找到密钥存储的位置打开.ssh后缀的文件夹打开.pub后缀的文件里面就是密钥，复制密钥在settings找到SSHandGPGkeys中创建密钥 在key中输入刚刚复制的密钥点击addSSHkey在打开gitbash 输入ssh -T git@github.com,在输入yes出现下面的界面就说明链接成功了 输入git confing --global user.name "这里输入用户名" 输入git confing --global user.email "这里输入注册github账号的邮箱" 克隆仓库 在github中打开创建的仓库，复制仓库的网址在文件夹里面创建一个boke文件夹然后复制这个文件夹的路径 打开git bash 输入cd / 刚刚复制的文件夹路径回车 在输入 gitclone后面接上刚刚复制的仓库网址成功后会出现一个和git里面仓库名字相同的文件夹 输入ls可以查看定位的文件夹里面的文件 输入 git add "这里面输入要上传的文件的名称" 输入 git commit -m "这里面输入你对刚刚上传的文件的备注" 在输入git push origin main 出现以下界面就说明成功了 在github中仓库中就可以看到文件
